# FidgetPix
Fidget App helps a fidgety person to remain calm and carry on.

## How to Use

1. Launch the app — a blank white canvas will appear.
2. Press any key on the keyboard — a colored stamp will appear at a random position on the canvas.
3. Each key is mapped to a specific image. Pressing the same key always stamps the same image, but at a different random location each time.
4. Press the **Delete** key to clear the canvas back to blank. Your key-to-image mappings are preserved, so you can keep stamping with the same assignments.

## Requirements

- macOS 14.0+
- Xcode 16+
